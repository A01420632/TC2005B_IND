CSS

¿Qué es CSS?

CSS "es un lenguaje {...} que nos permite agregar diseño y maquetación a nuestras páginas, y nos permite compartir esos estilos de un elemento a otro y de una página a otra" (Howe S., s.f).

Conceptos básicos

Selectores

Un selector designa a qué sección o elemento se le va a aplicar el estilo en CSS.
Los selectores pueden señalar un valor/atributo (id o clase por ejemplo) o un elemento (encabezado h1 o parráfo p por ejemplo).

Propiedades

Las propiedades son los atributos que se le asignan a los selectores.
Por ejemplo, si se quiere cambiar el color de un texto, se puede utilizar la propiedad "color".

Valores

Los valores son los atributos que se le asignan a las propiedades.
Por ejemplo, si se quiere cambiar el color de un texto a rojo, se puede utilizar el valor "red".

Ejemplo

p{
color: orange;
font-size: 16px;
}

Tipos de selectores

Selectores de tipo

Se pueden utilizar para cambiar el estilo de todos los elementos de un tipo en particular.
Entre los elementos que puede editar son: a, div, span, strong, em, h1-h6 y p.
Ejemplo:
p{
color: blue;
}

Selectores de clase

Se pueden utilizar para cambiar el estilo de todos los elementos que tengan una clase en particular.
Ejemplo:
.clase{
color: green;
}

Selectores de id

Se pueden utilizar para cambiar el estilo de un solo elemento que tenga un id en particular.
Ejemplo:
#id{
color: red;
}

Puntos de especificidad

Los puntos de especificidad son los que determinan cuál estilo se aplica a un elemento en caso de que haya conflictos.
Los selectores de id tienen más peso que los de clase, y los de clase tienen más peso que los de tipo.
Los puntos de especifidad se miden de la siguiente manera:

  • Selectores tipo: 0-0-1
  • Selectores clase: 0-1-0
  • Selectores id: 1-0-0
